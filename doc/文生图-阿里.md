# 阿里云万相图片生成接入笔记

> Provider：`aliyun`
> 模型类型：`image`
> 推荐模型：`wan2.7-image-pro`、`wan2.7-image`

## 后台配置

在 `/admin/models/image` 中配置：

| 字段 | 北京地域建议值 |
| --- | --- |
| Provider | `aliyun` |
| API 地址 | `https://dashscope.aliyuncs.com/api/v1` |
| 同步图片路径 | `/services/aigc/multimodal-generation/generation` |
| 异步图片路径 | `/services/aigc/image-generation/generation` |
| 查询路径 | `/tasks/{taskId}` |
| 调用模型名称 | `wan2.7-image-pro` 或 `wan2.7-image` |

如果需要覆盖端点，可在渠道配置 JSON 中写：

```json
{
  "endpoints": {
    "image": "/services/aigc/multimodal-generation/generation",
    "imageAsync": "/services/aigc/image-generation/generation",
    "imageQuery": "/tasks/{taskId}"
  }
}
```

## 地域注意

阿里云百炼不同地域的 API Key 和请求地址相互独立。北京、新加坡等地域不能混用 Key 与 Base URL。

新加坡地域通常使用：

```text
https://{WorkspaceId}.ap-southeast-1.maas.aliyuncs.com/api/v1
```

## 后端适配规则

前端传入：

```json
{
  "model": "wan2.7-image-pro",
  "prompt": "一间有着精致窗户的花店",
  "image": ["https://example.com/ref.png"],
  "size": "2K",
  "n": 1,
  "watermark": false
}
```

后端会适配为 DashScope 结构：

```json
{
  "model": "wan2.7-image-pro",
  "input": {
    "messages": [
      {
        "role": "user",
        "content": [
          { "image": "https://example.com/ref.png" },
          { "text": "一间有着精致窗户的花店" }
        ]
      }
    ]
  },
  "parameters": {
    "size": "2K",
    "n": 1,
    "watermark": false
  }
}
```

没有输入图片时可启用 `thinking_mode`。

## 响应解析

同步结果常见结构：

```json
{
  "output": {
    "choices": [
      {
        "message": {
          "content": [
            { "type": "image", "image": "https://dashscope-result...png" }
          ]
        }
      }
    ],
    "finished": true
  }
}
```

异步结果会先返回 `task_id`，后端会轮询 `imageQuery`，直到得到图片或超时。

## 参数建议

| 参数 | 说明 |
| --- | --- |
| `size` | `1K`、`2K`、`4K` 或具体像素。`wan2.7-image` 不支持 4K |
| `n` | 单次图片数量。关闭组图通常 1-4 |
| `thinking_mode` | 无参考图文生图时可开启 |
| `watermark` | 是否加水印 |
| `seed` | 随机种子 |

## 项目内规则

- 结果 URL 有效期通常为 24 小时，后端会立即转存到本地。
- 转存失败会导致生成失败并退款。
- 图片模型最低扣 1 金币。
- 上游错误可在 `/admin/error-logs` 查询。
