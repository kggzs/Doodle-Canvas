# 豆包 Seedream 图片生成接入笔记

> Provider：`doubao`
> 模型类型：`image`
> 推荐模型：`doubao-seedream-5-0-260128`

## 后台配置

在 `/admin/models/image` 中配置：

| 字段 | 建议值 |
| --- | --- |
| Provider | `doubao` |
| API 地址 | `https://ark.cn-beijing.volces.com` |
| API 路径 | `/api/v3/images/generations` |
| 调用模型名称 | `doubao-seedream-5-0-260128` |
| 用户显示名称 | `豆包 Seedream 5.0` |

默认参数示例：

```json
{
  "size": "2K",
  "response_format": "url",
  "sequential_image_generation": "disabled",
  "stream": false,
  "watermark": true
}
```

## 文生图

上游请求示例：

```bash
curl -X POST https://ark.cn-beijing.volces.com/api/v3/images/generations \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $ARK_API_KEY" \
  -d '{
    "model": "doubao-seedream-5-0-260128",
    "prompt": "星际穿越，黑洞，电影大片，超现实主义",
    "sequential_image_generation": "disabled",
    "response_format": "url",
    "size": "2K",
    "stream": false,
    "watermark": true
  }'
```

```json
  "stream": false,
```


## 后端适配规则

`providerType === "doubao"` 时，后端会发送：

```json
{
  "model": "doubao-seedream-5-0-260128",
  "prompt": "...",
  "size": "2K",
  "n": 1,
  "quality": "standard",
  "watermark": true,
  "image": "...",
  "sequential_image_generation": "disabled",
  "response_format": "url",
  "stream": false
}
```

## 响应与存储

后端会识别上游返回的 URL 或 Base64 图片，并转存为：

```text
/storage/generated/image/YYYY/MM/<uuid>.png
```

后台可在 `/admin/files` 和 `/admin/records` 追溯文件、模型、渠道、用户和项目。

## 注意事项

- API Key 使用火山方舟 Ark Key。
- 参考图建议使用服务端 `/storage/*` URL 或公网 HTTPS URL。
- 如果上游返回临时 URL，服务端会立即下载保存。
- 下载失败、MIME 不匹配或文件超限会导致生成失败并退款。
