# 阿里云万相视频生成接入笔记

> Provider：`aliyun`
> 模型类型：`video`
> 推荐模型：`wan2.7-i2v-2026-04-25`

## 后台配置

在 `/admin/models/video` 中配置：

| 字段 | 北京地域建议值 |
| --- | --- |
| Provider | `aliyun` |
| API 地址 | `https://dashscope.aliyuncs.com/api/v1` |
| 创建任务路径 | `/services/aigc/video-generation/video-synthesis` |
| 查询任务路径 | `/tasks/{taskId}` |
| 调用模型名称 | `wan2.7-i2v-2026-04-25` |

渠道配置 JSON：

```json
{
  "endpoints": {
    "video": "/services/aigc/video-generation/video-synthesis",
    "videoQuery": "/tasks/{taskId}"
  }
}
```

## 能力

万相 2.7 图生视频支持：

- 首帧生视频。
- 首尾帧生视频。
- 视频续写。
- 首帧 + 驱动音频。
- 多镜头叙事。
- 720P / 1080P。
- 2-15 秒整数时长。

## 后端适配规则

前端参数：

```json
{
  "model": "wan2.7-i2v-2026-04-25",
  "prompt": "人物缓慢转身看向镜头",
  "first_frame_image": "https://example.com/first.png",
  "last_frame_image": "https://example.com/last.png",
  "resolution": "720P",
  "dur": 5
}
```

后端会适配为：

```json
{
  "model": "wan2.7-i2v-2026-04-25",
  "input": {
    "prompt": "人物缓慢转身看向镜头",
    "media": [
      { "type": "first_frame", "url": "https://example.com/first.png" },
      { "type": "last_frame", "url": "https://example.com/last.png" }
    ]
  },
  "parameters": {
    "resolution": "720P",
    "duration": 5,
    "prompt_extend": true,
    "watermark": false
  }
}
```

请求头会包含：

```text
X-DashScope-Async: enable
```

## 查询任务

创建任务返回：

```json
{
  "output": {
    "task_id": "task_xxx",
    "task_status": "PENDING"
  }
}
```

查询成功返回：

```json
{
  "output": {
    "task_status": "SUCCEEDED",
    "video_url": "https://dashscope-result...mp4"
  }
}
```

后端会转存 `video_url` 到本地文件库。

## 输入素材组合

| 场景 | media 组合 |
| --- | --- |
| 首帧生视频 | `first_frame` |
| 首尾帧生视频 | `first_frame` + `last_frame` |
| 首帧配音频 | `first_frame` + `driving_audio` |
| 视频续写 | `first_clip` |
| 视频续写到尾帧 | `first_clip` + `last_frame` |

当前前端主要使用首帧/尾帧图片。更多素材类型可通过后续扩展节点参数透传。

## 注意事项

- 模型、API 地址和 API Key 必须属于同一地域。
- `task_id` 查询结果有效期通常为 24 小时。
- 输出视频 URL 有效期短，后端必须转存。
- Redis 中任务元数据默认保留 2 天。
- 转存失败会生成失败并退款。
